This Page Was Inspired By

Simple technologies that put in work. Having built the past few major projects using front-end SPA frameworks, I found it became increasingly difficult to not rely on a modern technology stack, even when it was an overengineered solution. I wanted to take a step back and, despite a personal website being a project I've been looking forward to building for a while, challenge myself to create one using nothing but the basics.
Creating this site was both a refreshing and rewarding experience. Taking advantage of what HTML and CSS have to offer, the result is a lightweight, static website with a clean, polished look.
